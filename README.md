# gapmonitor

SKHY/SKHX Hyperliquid 아비트라지 밴드 모니터.

`index.html` 하나로 동작하는 자기완결형 대시보드입니다. 브라우저에서 파일을 열면
Hyperliquid `info` API를 직접 호출해 SKHX(롱) / SKHY(숏) 간 %괴리·달러갭·Z-스코어·
변동성(σ) 밴드를 1분마다 갱신하고, 지정 지갑의 포지션과 재진입 트리거를 함께 표시합니다.

## 사용법

브라우저에서 `index.html`을 엽니다. 별도 빌드·의존성 없음.

## 지표

- **%괴리 / 달러갭** — `10·SKHY − SKHX` 기준 스프레드
- **Z-스코어** — 96봉(15m) 롤링 평균 대비 표준편차 배수, ±1σ 진입 / ±3σ 스톱
- **σ 하한 0.010** — 변동성이 이 아래면 "죽은장"으로 신호 무시
- **달러갭 백스톱 $505** — 리스크 상한 참조선
